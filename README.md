# YAGL (Yet Another Graphic Library)

Yagl is a simple and easy to use library for creating OpenGL based applications.

## Possibilities
 - Draw basic shapes
 - Render textures (Almost all existing formats)
 - Collision shapes
 - **SOON**: Audio
 - **SOON**: Shaders support
